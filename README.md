# 📚 Semantic Book Recommender

![App Banner](https://via.placeholder.com/1000x300/1e1e2d/6366f1?text=Semantic+Book+Recommender)

A sleek, robust, and modern ML machine-learning web application that dynamically recommends books using Collaborative Filtering. This project allows users to browse a dynamically generated wall of the **Top 50 Most Popular Books** based on aggregate rating metrics and then individually discover **intelligent recommendations** for any book they love based on cosine similarities calculated across thousands of user profiles.

## ✨ Features

- **Top 50 Books Showcase:** View the absolute highest-rated classic literature globally, calculated dynamically on page load based on our custom Popularity-based Recommender algorithm.
- **Collaborative Filtering Engine:** Search any book, ranging from *1984* to *Harry Potter*, and the ML algorithm instantly generates the top 4 most semantically similar books based heavily on User-Item rating overlaps.
- **High-Quality UI:** Built from scratch with glassmorphism, fluid interactive hover CSS transitions, beautiful gradients, and high-resolution cover integrations using the `Inter` font.
- **Robust Cover Integrity:** Intelligently fetches High-Res book covers from AWS affiliate sources via JS, gracefully handles missing 403 errors by falling back to medium-res, and lastly fails over to elegant Placeholders without distorting layout height.
- **Responsive Navigation:** Clean routing to a Contact Page.
- **Fast:** Deployed instantly via lightweight Flask.

## 🚀 Installation & Local Setup

**1. Clone the repository:**
```bash
git clone https://github.com/sahilgupta630/semantic-book-recommender.git
cd semantic-book-recommender
```

**2. Setup a Python Virtual Environment (Optional but Highly Recommended):**
It is extremely important to use Python 3.11 or lower to ensure legacy dataframe compatibility.
```bash
python -m venv venv
# For Windows
.\venv\Scripts\activate
# For Mac/Linux
source venv/bin/activate
```

**3. Install Dependencies:**
```bash
pip install -r requirements.txt
```

**4. Run the Engine!**
```bash
python app.py
```
*Head over to http://127.0.0.1:5000/ to view the project.*

## 🧠 Under the Hood (The Machine Learning)

The engine relies on 4 massive Pickle Objects `.pkl` holding trained, pre-processed Machine Learning Models data generated by `book-recommender-system.ipynb`.
1. **books.pkl**: Holds a complete cleaned dictionary referencing ISBN numbers, publication data, and High/Medium cover art image URLs.
2. **popular.pkl**: Pre-calculated Top 50 DataFrame aggregated from total vote count vs Average Rating metrics. 
3. **pt.pkl**: A meticulously generated Pivot Table with *Books on the Index* and *Users on the Columns*, replacing ratings with sparse matrices vectors to enable multi-dimensional calculations.
4. **similarity_scores.pkl**: Pre-calculated `cosine_similarity(pt)` resulting in a perfect Euclidean vector math correlation matrix of shape [706x706].
